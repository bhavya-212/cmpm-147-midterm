FANTASY NAME GENERATOR
-------------------------------------------

This generator creates fantasy names based on the parameters and a random background to go along with the names.

Generating names: The generator takes in predetermined prefixes, suffixes, and chunks to create a name.

Generating backgrounds: The generator takes in predetermined roles, traits, and goals to create a background.

Parameters (Users can choose in a dropdown)
-------------------------------------------
Category: Character/Place (User can choose whether they want a character name or place name to be generated)

Tone: Harsh/Elegant/Whimsical (User can choose how the name sounds based on three tones)

Length: Short/Medium/Long (User can choose how long they want the names to be generated)

Count: (User can enter any positive number for number of names/backgrounds they want to be generated)

-------------------------------------------
